# ParkingLotManagement

### 실행 방법

#### 깃 클론
<pre>
<code>
$ git clone https://github.com/sjoonb/ParkingLotManagement.git
</code>
</pre>

#### make
<pre>
<code>
$ make
</code>
</pre>

#### 실행
<pre>
<code>
$ ./ParkingLotManagement
</code>
</pre>
